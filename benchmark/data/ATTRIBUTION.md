# Attribution for COCO-derived subsets

This directory contains subsets of the COCO dataset (Lin et al., ECCV 2014).  
Original dataset: http://cocodataset.org  
License: Creative Commons Attribution 4.0 (CC BY 4.0)  
See LICENSE in this folder or LICENSES/CC-BY-4.0.txt.

## Modifications by PoseAdapt authors
- Selection of subsets for continual learning benchmarks.
- New benchmark splits (BodyParts, Domain-Incremental, etc.).
- Additional annotations for incremental skeletons.
- Preprocessing (cropping, resizing, normalization).

## Attribution requirement
If you use these subsets, you must:
1. Cite the COCO dataset paper:
   - Tsung-Yi Lin et al. "Microsoft COCO: Common Objects in Context." ECCV 2014.
2. Credit "COCO dataset" in your work.
3. Indicate that subsets/modifications were prepared by the PoseAdapt authors.

These subsets remain licensed under CC BY 4.0.
